# Student Performance Analysis Dashboard using Excel

## Project Overview

This project analyzes student performance using Microsoft Excel. The dashboard provides insights into attendance, study hours, assignment scores, and exam performance.

## Features

* Student Count KPI
* Average Score KPI
* Gender-wise Performance Analysis
* Attendance Analysis
* Top Performing Students
* Interactive Pivot Tables and Charts

## Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Data Visualization

## Dataset

50 student records with:

* Gender
* Attendance
* Study Hours
* Assignment Scores
* Mid Exam Scores
* Final Exam Scores
* Average Scores
